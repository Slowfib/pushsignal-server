PushSignal REST Service Help

Account

Create

PurposeTo create a new PushSignal account. This is the only call that does not require HTTP authentication.
URLhttp://PushSignal.com/api/account/create
MethodPOST
Querystringemail=User's email address.
name=User's name.
description=A short description of up to 255 characters.
Returns200 OK & XML
400 Bad Request

Register Device

PurposeTo register a new device to the logged-in account.
URLhttp://PushSignal.com/api/account/registerDevice
MethodPOST
QuerystringdeviceType=[C2dm|Mqtt]
deviceId=Phone unique identifier; for Android devices this represents the android.provider.Settings.Secure.ANDROID_ID value.
registrationId=Push service registration id. For C2DM this represents the value returned by the registration call.
Returns200 OK & XML
400 Bad Request
401 Unauthorized
404 Not Found
